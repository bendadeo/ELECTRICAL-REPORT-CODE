This project is for the electrical report.
In oder to run the system you need to following step.
1. Upload the RC522bare to the board and the connection need to follow this oder:
   D10 - SDA
   D13 - SCK
   D11 - MOSI
   D12 - MISO
   GND -GND
   D9 - RST
   3V3 - 3V3
2. Run the file electrical report to open the platform and fill out the UID card, Student Id and Name. Please remember that each section need to follow the oder
3. Run the file sosanh.py, remember to change the file location let the python know where is your database. And finishing.
